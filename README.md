# cybersecurity-task-4
Password Security &amp; Authentication Analysis
## Objective
The objective of this task is to understand how passwords are stored, how weak passwords can be attacked using wordlists, and how strong authentication mechanisms protect systems from unauthorized access.

This task was performed in a **legal and controlled lab environment** using self-created test passwords and hashes.

---

## Tools Used
- Kali Linux
- Hashcat
- John the Ripper
- rockyou.txt wordlist
- Nano text editor
- Terminal

---

## Concepts Covered
- Hashing vs Encryption
- Common hash types (MD5, SHA-1, bcrypt)
- Dictionary (wordlist) attack
- Brute-force attack
- Weak vs strong passwords
- Multi-Factor Authentication (MFA)
